## 🧭 Choose Your Path

> **Not sure where to start?** Pick your role to get personalized recommendations.

### 👤 Site Administrator
*"I want to install, configure, and manage a XOOPS site."*

→ Start with [Getting Started](../01-Getting-Started/Getting-Started.md) for installation and setup, then explore [Admin Panel Overview](../01-Getting-Started/First-Steps/Admin-Panel-Overview.md) to learn site management.

### 🔧 Module Developer
*"I want to build modules for XOOPS."*

→ Begin with [Hello World Module](../03-Module-Development/Tutorials/Hello-World-Module.md), then learn [XOOPS Architecture](../02-Core-Concepts/Architecture/XOOPS-Architecture.md) and [XMF Framework](../05-XMF-Framework/XMF-Framework.md) for modern development.

### 🎨 Theme Developer
*"I want to create or customize themes."*

→ Master [Smarty Templating](../02-Core-Concepts/Templates/Smarty-Basics.md) first, then dive into [Theme Development](../02-Core-Concepts/Themes/Theme-Development.md).

### 🚀 Core Contributor
*"I want to contribute to XOOPS core development."*

→ Explore [XOOPS 4.0 Roadmap](../07-XOOPS-4.0/XOOPS-4.0-Roadmap.md), review [Hybrid Mode Contract](../07-XOOPS-4.0/Specifications/Hybrid-Mode-Contract.md) for compatibility guarantees, and follow [Contributing Guidelines](../09-Contributing/Contributing.md).
